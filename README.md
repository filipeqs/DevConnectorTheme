# DevConnector Theme / UI 

Full theme with Sass (.scss) files

---

## Instructions

First clone this repository.

```bash
$ git clone https://github.com/filipeqs/dev-connector-theme.git
```

Open index.html

```bash
dist/index.html
```

## Features

- Landing Page
- Register/Login Form
- Profile, Experience & Education Pages
- List of Developers Page
- Profile Info Page
- List of Posts Page
- Post & Comments Form
